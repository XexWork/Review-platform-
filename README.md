# Review-platform-

## Overview
This is a platform created for posting products and leaving reviews about them. The platform requires registration and account login.

## Features
- **Account registration option**: Users can register an account and then log in!
- **Ability to list products**: Users can list their products on the platform!
- **Ability to leave reviews**: Users can leave reviews on the platform!
- **Calculating the average rating**: The platform calculates the average rating of the user's product!

## Installation
### Requirements
Python 3.8+   
sqlalchemy   
fastapi   
passlib   
pydantic   

### Setup
1. Clone the repository:
   ```sh
   git clone https://github.com/XexWork/Review-platform-
   cd Review-platform-
   ```
2. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
5. Run:
   ```sh
   uvicorn main:app --reload
   ```
